# .quality is a collection configuration files for linters and code analysis tools. 
Collectcion of configurations dotfiles for code quality tools.

#### JavaScript
* [Eslint](https://github.com/eslint) - [.eslintrc.yml](https://github.com/khusnetdinov/.quality/blob/master/.eslintrc.yml)

#### Ruby
* [Brakeman](https://github.com/presidentbeef/brakeman) - [.brakeman.yml](https://github.com/khusnetdinov/.quality/blob/master/.brakeman.yml)
* [Haml](https://github.com/brigade/haml-lint) - [.haml-lint.yml](https://github.com/khusnetdinov/.quality/blob/master/.haml-lint.yml)
* [Reek](https://github.com/troessner/reek) - [.reek](https://github.com/khusnetdinov/.quality/blob/master/.reek)
* [Rubocop](https://github.com/bbatsov/rubocop) - [.rubocop.yml](https://github.com/khusnetdinov/.quality/blob/master/.rubocop.yml)
* [Scss](https://github.com/brigade/scss-lint) - [.scss-lint.yml](https://github.com/khusnetdinov/.quality/blob/master/.scss-lint.yml)
